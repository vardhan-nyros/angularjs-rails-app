== README

This is an AngularJS + Rails application used Rails version 4.x, and Ruby version 2.x.

The following are the AngularJS features which I have used in this application:

* Angular JS controllers, Services and Route Provides

* CRUD operations using angular controller and route providers

* All views are integrated with bootstrap

* Integrated pagination in Angular+Rails using will_paginate

* Integrated Google Map

* Configured Filters and sortings of AngularJS
